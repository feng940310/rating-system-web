# 本地视频存储目录

此目录用于存储项目中的视频文件。

## 文件夹结构
```
static/videos/
├── works/           # 作品视频
│   ├── 1/          # 作品ID为1的视频
│   ├── 2/          # 作品ID为2的视频
│   └── ...
├── demo/           # 演示视频
├── thumbnails/     # 视频缩略图
└── README.md       # 说明文档
```

## 支持的视频格式
- MP4 (推荐)
- WebM
- OGV

## 文件命名规范
- 主视频: `main.mp4`
- 缩略图: `thumbnail.jpg`
- 封面图: `cover.jpg`

## 注意事项
1. 视频文件不要过大，建议单个文件不超过50MB
2. 使用H.264编码以确保最佳兼容性
3. 分辨率建议不超过1080p
4. 添加到.gitignore以避免上传到代码仓库（如果文件过大）
